Odoo-11 Server ကို စတင်ခြင်းနှင့် ရပ်ဆိုင်းခြင်း

odoo server folder အတွင်းသို့၀င်ရောက်ပါ။
ဥပမာ။ cd /home/username/odoo/ ပြီးလျှင် ./odoo-bin ဟုရိုက်ပြီး စတင်လိုက်ပါ။
Browser ကို ဖွင့်ပြီး localhost:8069 ဟုခေါ်လိုက်ပါ။ odoo server default port သည် 8069 ဖြစ်သည်။
./odoo-bin နောက်တွင် တွဲ၍သုံးနိုင်သော options များရှိသည်။ development အတွက် အသုံးများသော options များကိုအောက်ပါအတိုင်းဖော်ပြပေးလိုက်ပါသည်။

  • --xmlrpc-port=port_no (port no ပြောင်းသုံးရာတွင်သုံးသည်။)
  • -d DB_NAME, --database=DB_NAME (database ကိုသတ်မှတ်ပြီး run ရာတွင်သုံးသည်။)
  • -c CONFIG, --config=CONFIG (config file သတ်မှတ်ထားပါက ခေါ်၍သုံးနိုင်သည်။)
  • -u module_name (code modify လုပ်ပြီးသောအခါ module ကို update လုပ်ရာတွင်သုံးသည်။ -d databasename နှင့်တွဲသုံးနိုင်သည်။)
  • အခြားသော options များကို ./odoo-bin --help ဟု run ၍ ဆက်လက်လေ့လာနိုင်သည်။
